# Todo API (Minimal API • .NET 9)

A simple Todo API built with **ASP.NET Core Minimal APIs**, **Entity Framework Core**, and an **In-Memory Database**.  
This project demonstrates clean minimal API structure using **route groups**, CRUD operations, and built-in Swagger documentation.

---

## Features

- ✔ ASP.NET Core **Minimal API** (no controllers)
- ✔ Full **CRUD** operations for Todo items
- ✔ **Entity Framework Core** with InMemory database
- ✔ Organized routing using **Route Groups**
- ✔ Interactive API documentation using **Swagger / NSwag**
- ✔ Fully asynchronous API
- ✔ Lightweight and easy to extend

---

## Technologies Used

- **.NET 9**
- **ASP.NET Core Minimal API**
- **Entity Framework Core**
- **EF Core InMemory Provider**
- **NSwag (OpenAPI/Swagger)**
- **C# 13**
